# Securosys HSM API Examples

This repository contains various code samples for how to use the
[Securosys Primus HSM](https://www.securosys.com/en/hsm/hsm-overview)
and [Securosys CloudHSM](https://cloud.securosys.com/cloudhsm)
in various applications and programming languages.

For details, please see:

- The README of each directory for usage instructions.
- The [documentation site](https://docs.securosys.com/).
